# MetroVSz城市交通服务水平可视化系统
个人毕业设计, web端的MetroVSz系统, 实现了公交服务水平的可视化

## 技术栈
> vue全家桶 + d3.js + bootstrap + 高德地图API for javascript + h5 + js + css + axios

## 安装
需要先注册个人开发者账号，修改config.js中key的值
```
npm install
```

### 开发环境下编译
```
npm run serve
```

### 生产环境下编译
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
